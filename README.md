# 🧮 Math Quiz Game

A simple console-based Math Quiz game built with Python.

In this game, players answer 10 random math questions within a limited time. At the end of the game, the final score and overall performance are displayed.

---

## 📖 Overview

This project was created to practice core Python programming concepts, including:

- Functions
- Loops
- Conditional statements
- Error handling
- Time management
- Random number generation

---

## ✨ Features

- Random math questions
- Four arithmetic operators (+, -, *, /)
- 10-second time limit for each question
- Final score summary
- Input validation
- Console-based user interface
- Countdown before the game starts

---

## 🚀 How to Run

1. Make sure Python 3 is installed on your computer.
2. Clone this repository or download the project files.
3. Open a terminal in the project folder.
4. Run the following command:

bash
python main.py


5. Press ENTER to start the game and enjoy!
